# ZQFlexibleToolExample

Demo app for `ZQFlexibleTool`.

## Run

1. Generate or open `Example/ZQFlexibleToolExample.xcodeproj`.
2. Run `pod install` in `Example/`.
3. Open the generated workspace and run `ZQFlexibleToolExample`.

## Demo pages

- Home: links to the rest of the demo pages
- Navigation: back handling, hide/show navigation bar
- File Manager: save, read, delete sample file
- Permissions: request camera and photos, open Settings
- Daily Tool: string, array, date, color and VC helpers
